Projects

Scale Degree Practice Tool

TRY IT OUT!

I built this application for my own use - but I also wanted to build something using CSS Grid. The whole application was built using only HTML, CSS and Vanilla JS. The application is currently responsive for medium-larger monitors and smartphones. I will be making an update for macbook/laptop size screens soon!

For the non-musician it is difficult to understand what this application does and would require a face to face explanation -- but essentially it is a practice tool that flashes a random key (the letter name selections) with a random scale degree (the roman numeral selections). These two random selections tell the user a specific chord they need to play before the next one flashes. The goal is that the user will eventually be able to play any possible combination very quickly without much thought - which is an important skill for a jazz musician. The logic only uses the selected options to randomize - so it allows the user to start in a comfortable spot and slowly add to build into mastering the exercise

I built this application to help me reach a personal goal - and it definitely helped me meet it!
